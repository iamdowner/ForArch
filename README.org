* Install

This will link user-wide configurations folder to your =$HOME=
#+begin_src bash
stow -d src home -t ~/
#+end_src

This will link system-wide configurations folder to your =$HOME=
#+begin_src bash
sudo stow -d src etc -t /etc
#+end_src
